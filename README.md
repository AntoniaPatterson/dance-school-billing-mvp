# Dance School Billing Automation (MVP)

This project is a cloud-native, admin-only MVP designed to reduce manual billing work for dance school owners/administrators.

It focuses on solving a small set of high-friction billing tasks that are currently time-consuming and error-prone.

## Core Problems Solved
- Bulk class price updates
- One-off custom fees (e.g. late payment fees)
- Bulk overdue invoice automation and reminders

## MVP Scope
This is a limited MVP, The following is out of scope:
- Parent portal
- Payment processing
- Class booking
- Attendance tracking
- Discounts
- Waiting list
